Displaying Product Items Using JavaScript's `map` Method

In this tutorial, we will learn how to use JavaScript's `map` function to display a list of product items with their titles, categories, and prices on a webpage. This is typically how we would handle data received from a backend API in JSON format and render it dynamically on the frontend.

When working with arrays, objects are often the most common type of element you'll encounter. When fetching data from a third-party source, such as an API, you typically receive more data than necessary for your specific use case. The map method is particularly useful in such scenarios as it allows you to selectively extract and transform only the data you intend to display on your application's screen. Notably, the map method always returns a new array based on the transformations applied to each element during iteration. In environments like JSX, commonly used in React applications, the map function's versatility shines through, enabling developers to render lists of diverse data types seamlessly.

1. Setting Up the HTML Structure

First, let's set up the basic HTML structure with a title and a container to display our product items: 

<!DOCTYPE html>
<html lang="en">
  <head>
    <meta charset="UTF-8" />
    <meta name="viewport" content="width=device-width, initial-scale=1.0" />
    <title>Product Items Display</title>
    <link rel="stylesheet" href="mapFunction.css" />
  </head>
  <body>
    <div class="container">
      <div class="list-div">
        <h1>Product Items</h1>
        <div id="myDiv"></div>
      </div>
    </div>
  </body>
</html>

2. Adding the JavaScript for Product Display

Next, let's add JavaScript to dynamically display the product items inside the `myDiv` container: 

<script>
  // Define an array of product items
  const productItems = [
    {
      title: "Washing Machine",
      category: "Electronics",
      price: 1500,
    },
    {
      title: "Juicer",
      category: "Electronics",
      price: 500,
    },
    {
      title: "T-shirt",
      category: "Apparel",
      price: 150,
    },
    {
      title: "Jug and Glass set",
      category: "Household",
      price: 250,
    },
  ];

  // Select the div where products will be displayed
  const divElement = document.getElementById("myDiv");

  // Use map function to create elements for each product item
  productItems.map((item) => {
    const itemDiv = document.createElement("div");
    const itemTitle = document.createElement("p");
    const itemCategory = document.createElement("p");
    const itemPrice = document.createElement("p");

    // Set text content for each element
    itemTitle.textContent = `Title: ${item.title}`;
    itemCategory.textContent = `Category: ${item.category}`;
    itemPrice.textContent = `Price: $${item.price}`;

    // Append elements to the item div
    itemDiv.append(itemTitle);
    itemDiv.append(itemCategory);
    itemDiv.append(itemPrice);

    // Append item div to the main container div
    divElement.append(itemDiv);
  });
</script>

3. Explanation of the Code

Here, we'll explain what each part of the code does and how it achieves our goal of displaying product items:

  • Defining Product Data

    We start by defining an array `productItems` that contains objects. Each object represents a product with properties like `title`, `category`, and `price`.

  • Creating HTML Elements Dynamically

    Using JavaScript's `map` function, we iterate over each `item` in `productItems`. For each `item`, we create a `div` (`itemDiv`) and `p` elements (`itemTitle`, `itemCategory`, `itemPrice`) to display the product's details.

  • Setting Text Content

    We set the text content of `itemTitle`, `itemCategory`, and `itemPrice` to display the `title`, `category`, and `price` of each product item, respectively.

  • Appending Elements to the DOM

    Finally, we append `itemDiv` containing the product details to `myDiv` in the HTML, which dynamically renders each product item on the webpage.
