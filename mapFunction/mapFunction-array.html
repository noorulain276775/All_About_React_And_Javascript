Square Root of Numbers Using the `map` Function

In this tutorial, we will learn how to use JavaScript's map function to calculate the square roots of an array of numbers and display them in an HTML list.

What is JavaScript map method and Why it is used

JavaScript map method is used to transform a list, and a list is a simple collection of elements

  • map() creates a new array from calling a function for every array element.
  • map() does not execute the function for empty elements.
  • map() does not change the original array.

This is all about map function.

Now below is the complete Guide how to use map method

1. Setting Up the HTML Structure

First, we set up a basic HTML structure with a title, a heading, and a container for our list: 

<!DOCTYPE html>
<html lang="en">
  <head>
    <meta charset="UTF-8" />
    <meta name="viewport" content="width=device-width, initial-scale=1.0" />
    <title>Map Function Tutorial</title>
  </head>
  <body>
    <div>
      <div class="list-div">
        <h1>Square Root of Numbers Using map Function</h1>
        <ul id="myList"></ul>
      </div>
    </div>
  </body>
</html>

2. Adding the JavaScript

Next, we add a <script> tag to include our JavaScript code that performs the calculations: 

<script>
  const numbers = [4, 5, 6, 7, 8];
  const listElement = document.getElementById('myList');

  numbers.map((num) => {
    const listItem = document.createElement('li');
    listItem.textContent = Math.sqrt(num);
    listElement.append(listItem);
  });
</script>

This script does the following:

  • Defines an array of numbers.
  • Selects the <ul> element with the ID myList.
  • Uses the map function to iterate over each number in the array.
  • Calculates the square root of each number using Math.sqrt().
  • Creates a new <li> element for each result.
  • Sets the text content of each <li> element to the square root value.
  • Appends each <li> element to the <ul>.
